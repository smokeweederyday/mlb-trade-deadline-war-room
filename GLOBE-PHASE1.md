# Boring Bets Globe — Phase 1

This phase is isolated from the active season enrichment process.

Included:
- draggable, zoomable orthographic Earth
- clickable MLB stadium markers for the selected date
- venue-local clocks and local game times
- games list linked to Game Center
- permanent venues.json schema for coordinates, timezone, altitude, field bearing, roof, surface, and dimensions
- build_globe_data.py to refresh venue metadata from MLB
- weather-ready and travel-ready UI slots

Future phases attach live precipitation, wind relative to field orientation, stadium geometry, team routes, planes, and park-factor intelligence to the same venue objects.
