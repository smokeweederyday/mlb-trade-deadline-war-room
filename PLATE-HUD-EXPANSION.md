# Plate HUD Expansion

- Keeps the embedded Live Game Center plate scene compact.
- Clicking the plate scene opens the immersive full-screen retro ghost Plate View HUD.
- Uses the approved Tron-style ballpark concept as the default park artwork.
- Escape, close, and backdrop click return to the normal Live Game HUD.
- Preserves current game context in a small live ribbon without altering the approved default art.
