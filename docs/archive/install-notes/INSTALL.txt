BORING BETS — LIVE DASHBOARD PHASE 3

1. Unzip this package.
2. Copy everything inside BORING-BETS-LIVE-DASHBOARD-PHASE3 into the root of the current boring-bets project.
3. Choose Merge/Replace when Finder asks.
4. From the boring-bets folder run:

   python3 -u scripts/check_live_dashboard_phase3.py

5. Start the local server:

   python3 -m http.server 8000

6. Open:

   http://localhost:8000/live.html
